# SOP-EL-07 — Over-current (O/C) Feeder Trip
1. Confirm relay flag and disturbance-recorder (DR) record.
2. Patrol the line for visible fault (broken conductor, tree contact, kite/bird).
3. One test-charge attempt is permitted after patrol confirms no visible fault.
4. If the feeder holds, restore load in blocks of <=30%.
5. If it re-trips, isolate the affected section and escalate to the Assistant Engineer (AE).
